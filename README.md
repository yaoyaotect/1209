* 下载
    ``` 
     git clone https://code.aliyun.com/task2016/allcommand.git
    ```
* 进入目录
    ``` 
     cd allcommand
    ```
* 安装依赖
    ``` 
     npm install 
    ```
* 启动项目
    ``` 
     node app.js
    ```
* 监听
    ``` 
     gulp watcher
    ```
* 访问
    ``` 
     http://127.0.0.1:9805/loading.html
    ```
